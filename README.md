#  MetroBaz

This app allows the user to display the location of the free Metro Link Nihonbashi, Metro Link Nihonbashi E-Line Buses and the Marounouchi and Hibiya Shuttle buses of Tokyo.

In addition to the routes and the bus locations, **MetroBaz** provides bus time table at every bus stops and the nearest bus stop from the user location in an intuitive user interface.

## Highlights

**MetroBaz** provides the following functionality:

* Supports 4 different bus lines, Nihonbashi, Ningyocho, Marunouchi and Hibiya.
* Displays the current location of all buses of the selected line.
* Displays all bus stops and the times for the next 3 buses.
* Calculate the distance to the nearest bus stop and the times for the next 3 buses.
* Supports different map types, Standard, Hybrid and Satelite.
* Supports both English and Japanese languages.

The app requires no account, registration, subscription, payment information or advertising. Users simply starts the app and immediately receive the information they need.

The intentionally simple interface is a usability and accessibility choice.
